```markdown
# Contributing to AlgoViz

Thanks for wanting to help!

## How to Contribute

1. Fork this repo.  
2. Clone your fork.  
3. Create a new branch.  
4. Make your changes and commit.  
5. Push your branch.  
6. Open a pull request.

## Code Style

- Write clean and simple code.  
- Add comments if needed.  
- Put new algorithms in `src/algorithms/`.

## Commit Messages

Use a short message like:  
`feat: add new algorithm`  
`fix: fix a bug`

## Questions?

Open an issue or ask anytime.
```
